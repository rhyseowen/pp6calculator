33 ## Usage
34 
35 
36 On running the program you are prompted to select a menu. Operations are sorted by the week they were featured in the [c++ course](http://www2.warwick.ac.uk/fac/sci/physics/research/epp/resources/teaching/software_development_2014/)
37 
38 Currently Available menus are:
39 
40  1) weekone : week one (arithmatic)
41  2) weektwo : week two (data operations)
42 
43 ### Week One: Arithmatic
44 
45 Within the week one menu available operations are:
46 
47  1) + : Simple addition of two numbers
48  2) - : Simple subtraction of two numbers
49  3) / : Simple division of two numbers
50  4) * : Simple multiplication of two numbers
51  5) solveAxis : Find the x-intercept of a straigt line
52  6) solveQuadratic : find the real roots of a quadratic equation
53  7) 3VectorLength : find the length of a 3 vector
54  8) 4VectorLength : find the length of a 4 vector
55  9) invarientMass : find the invarient mass of a momentum 4 vector
56  h) help : Show this message
57  u) up : Go up one menu level
58  q) quit : exit the calculator
59 
60 ### Week Two: Data Operations
61 
62 Within the week two menu available operations are:
63 
64  1) swap : swap 2 numbers
65  2) sort : sort 8 numbers
66  3) vectorSort : sort 8 numbers (using a vector instead of an array)
67  4) generateMuons : Generate 100 random muons
68  5) readInMuons : Read in muons from file
69  h) help : Print this Message
70  u) up : Go up one menu level
71  q) quit : exit the calculator
72 
73 ### Return Codes
74 If the program encounters an error and is subsequently quit it will return one of the following codes.
75 
76  0 - Success
77  2 - Operator not recognised
78  3 - Divide by 0
79  4 - Input File contains too many muons
