43 ## Usage
44 
45 
46 On running the program you are prompted to select a menu. Operations are sorted by the week they were featured in the [c++ course](http://www2.warwick.ac.uk/fac/sci/physics/research/epp/resources/teaching/software_development_2014/)
47 
48 Currently Available menus are:
49 
50  1) weekone : week one (arithmatic)
51  2) weektwo : week two (data operations)
52 
53 ### Week One: Arithmatic
54 
55 Within the week one menu available operations are:
56 
57  1) + : Simple addition of two numbers
58  2) - : Simple subtraction of two numbers
59  3) / : Simple division of two numbers
60  4) * : Simple multiplication of two numbers
61  5) solveAxis : Find the x-intercept of a straigt line
62  6) solveQuadratic : find the real roots of a quadratic equation
63  7) 3VectorLength : find the length of a 3 vector
64  8) 4VectorLength : find the length of a 4 vector
65  9) invarientMass : find the invarient mass of a momentum 4 vector
66  h) help : Show this message
67  u) up : Go up one menu level
68  q) quit : exit the calculator
69 
70 ### Week Two: Data Operations
71 
72 Within the week two menu available operations are:
73 
74  1) swap : swap 2 numbers
75  2) sort : sort 8 numbers
76  3) vectorSort : sort 8 numbers (using a vector instead of an array)
77  4) generateMuons : Generate 100 random muons
78  5) readInMuons : Read in muons from file
79  h) help : Print this Message
80  u) up : Go up one menu level
81  q) quit : exit the calculator
82 
83 ### Week Three: Object Orientation
84 
85 Within the week three menu available operations are:
86 
87  1) interval : Calculate the interval of a 4 vector
88  2) boost_z : boost a four vector along the z axis
89  3) generateMuons: Generate Muons and use the particle class
90  h) help : Print this Message
91  u) up : Go up one menu level
92  q) quit : exit the calculator
93

95 ### Return Codes
96 If the program encounters an error and is subsequently quit it will return one of the following codes.
97 
98  0 - Success
99  2 - Operator not recognised
100  3 - Divide by 0
101  4 - Input File contains too many muons
